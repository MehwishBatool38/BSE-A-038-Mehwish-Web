🎓 EduFee — Fee Management System
📋 Overview

EduFee is a web-based fee management system designed for educational institutions to efficiently manage student fees. It provides a simple interface for administrators and students to handle fee plans, payments, and tracking.

🚀 Features
Admin Module
Manage students (add, edit, delete)
Create and manage fee plans (installments supported)
Assign fee plans to students
Record and track payments
View payment history and receipts
Monitor pending dues
Send notifications
Student Module
View assigned fee plans
Track payment status and dues
View payment history
Download fee receipts
Receive notifications
🛠️ Technologies Used
HTML
CSS
JavaScript (Vanilla JS)
LocalStorage (for data handling)
