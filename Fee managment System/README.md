🎓 EduFee — Fee Management System

EduFee is a simple and efficient Fee Management System designed to manage student fees, track payments, and generate receipts. It provides separate interfaces for administrators and students with a clean and responsive user interface.

📋 Overview

The system enables educational institutions to manage fee operations digitally, including fee planning, assignment, and payment tracking using a single-page web application.

🚀 Features
Administrator
Manage student records
Create and manage fee plans
Assign fee plans to students
Record payments
Track fee status
Generate receipts
Student
View fee details
Check pending dues
View payment history
Access receipts
Manage profile
🛠️ Technology Stack
Frontend: HTML5, CSS3, JavaScript
Storage: LocalStorage
